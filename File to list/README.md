# Projeto: Lista de Tarefas (To-do List)

## Sobre o projeto
Este é um projeto simples para ajudar os iniciantes a aprenderem desenvolvimento web. Com ele, você pode adicionar, marcar como concluída e remover tarefas do dia a dia. O objetivo é mostrar habilidades básicas em HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar novas tarefas
- Visualizar tarefas cadastradas
- Marcar tarefas como concluídas
- Remover tarefas
- Salva automaticamente as tarefas no navegador (LocalStorage)
- Interface fácil e responsiva

## Como rodar o projeto
1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.

Ou acesse a versão online (GitHub Pages): **[Seu link aqui](#)**
