# Projeto: Lista de Tarefas (To-do List)

Olá! Meu nome é **Falcaofe** e este é um dos meus primeiros projetos para o portfólio na área de tecnologia.

## Sobre o projeto

Desenvolvi esta aplicação como parte dos meus estudos em desenvolvimento web.  
Ela permite adicionar, visualizar, marcar como concluída e remover tarefas do dia a dia, de maneira prática e intuitiva.  
O objetivo deste projeto é aprender e demonstrar habilidades básicas com HTML, CSS e JavaScript, além de mostrar minha vontade de evoluir cada vez mais na área.

## Funcionalidades

- Adicionar nova tarefa
- Visualizar lista de tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Salvar automaticamente as tarefas no navegador (LocalStorage)
- Interface fácil e responsiva

## Como executar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

- Filtro para tarefas concluídas e pendentes
